# Resume Optimization Prompt- Case Study

## Overview

This project focuses on designing and improving a prompt that helps optimize a resume for Prompt Engineering ans Generative AI roles. The objective was to create a prompt that generates practical, accurate, and professional resume improvements while keeping the information truthful.

---
## Why I Built This Project 

I built this project because I wanted to learn Prompt Engineering by solving a real-world problem instead of only reading about it.

As someone with a Computer Science degree but limited professional experience in AI, I wanted to create practical projects that demonstrate my learning journey. I also wanted to build a GitHub portfolio that i can confidently share with recruiters to showcase my Prompt Engineer skills.

Working on this project helped me become more comfortable using GitHub, documenting my work, and designing prompts with clear objectives and constraints.

## My Approach 

I started with a simple prompts and gradually improved it by adding :

 - A specific role(Prompt Engineer / Gnerative AI)
 - Clear objectives
 - Constraints to prevent false information
 - A structured output format
 - Multiple prompt iterations based on testing

This process helped me understand how small improvements in a prompt can significantly imrpove the quality of AI-generated responses.

---

## Results

After tseting the prompt, I observed that ChatGPT:
 - Improved the formatting and readability of my resume.
 - Sugeested stronger wording for my professional summary.
 - Recommended including GitHub projects to better demonstrate practical experience.
 - Produced more relevant resposes when given clear context and instructions.

---

## Key Takeaways

This project taught me that Prompt Engineering is not simple asking AI to complete task. It involves defining a clear objective, providing suffiecient context, setting constraints and cintinuously refining prompt based on the results.

More importantly, I realized that building real project is the best way to learn Prompt Engineering . Each project improves both my technical skills and my confidence to demonstrate those skills to recruiter.
