# Prompt Iteration

## Objective

The goal was to design a prompt that generates a personalized cover letter based on an applicant's resume and a specific job description

---

## Version 1

### Prompt
>How do i write a professional cover letter?

### Observartion

The response provided by general advice and best practices but did not generate a personalized cover letter.

---

## Version 2

### Prompt 
> Write a professional cover letter using my resume and the provided job description.

### Observation

The generated cover letter was relevant to the job description but still include generic wording and lacked clear instructions about tone, structure and accuracy.

---

## Version 3

### Prompt
> Act as an experienced career cover coach and professional resume writer .
>
> Using my resume and the provided job description , write a professional cover letter that:
>
> - explains why I am strong fit for the role
> - highlights only the most relevant skills and acheivements
> - maintain a professional and confident tone.
> - doesnot invent experience or qualifications
> - incorporates relevant keywords from the job description where appropriate
> - keep the letter between 250-350 words
> - ends with a professional closing that encourages further description

### Why this version performed better

This version produced more focused and personalized results by clearly defining the AI's role, providing relevant context, and specifying output constraints. The cover letter aligned more closely with the jobs requirements while accurately representing the applicant's experience.
