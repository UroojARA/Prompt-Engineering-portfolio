#  AI Cover Assistant

##  Overview 

AI career Assistant is a prompt engineering project designed to help job seekers throughout the application process . This repository desmonstrates how carefully designedprompts can generate professional career documents, optimize LinkedInn profiles and assist with recruiter communication.

The project showcases prompt engineering techniques such as prompt iteration, role prompting, context injection and structured output genearation.

---

## Features

 - Resume Trailoring
 - Cover Letter Generation
 - LinkedIn Headline Optimization
 - LinkedIn "About" Section Writing
 - Recruiter Outreach Message
 - Professional Email Drafting
---

## Prompt Engineering Techniques
 - Role Prompting
 - Context-Aware Prompting
 - Prompt Iteration
 - Constratint-Based Prompting
 - Structured Output
